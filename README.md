#说明：
1、直接访问127.0.0.1:8000

3、然后点击用户注册可以注册用户
4、点击用户管理会会调到用户管理页面
5、点击服务器列表2，可以查看服务器列表，还可以远程对服务器使用命令（可能有的没装上getone，就不能操作）
6、点击服务器列表1，只能查看服务器列表
7、点击数据库会出现一个竖状的图标
8、直接在浏览器上输入http://127.0.0.1:8000/user/login/ 也是用户的登录页面
9、request_script里面的脚本是配合api做get、post请求的
